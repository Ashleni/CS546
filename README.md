# Delicacy - CS546 Final Project
## About Delicacy
Delicacy is designed to bring official NYC restaurant inspection data and authentic user reviews together to provide a more reliable and transparent restaurant hygiene system. Most food applications are based on food reviews rather than hygiene. Delicacy displays a given restaurant's historic letter grade and violation information. Users can leave reviews and comments on the establishment indicating real-time hygiene conditions, with the option to add photos as well. Delicacy puts food safety into the hands of the community for more accurate, frequent, and better reporting.


## Getting Started

### Installation

Some Basic Instructions on how to download and run this project

1. Download and unzip project file 
2. Run npm install to aquire all nessesary packages
3. Run npm run seed to populate the database 

### Seed File

```
npm run seed
```

Over 2000 real NYC restaurants have been scraped from the following dataset:

https://data.cityofnewyork.us/Health/DOHMH-New-York-City-Restaurant-Inspection-Results/43nn-pn8j/about_data 

Running the seed file will populate Delicacy with these restaurants. 


### Starting the Website

```
npm start
```
